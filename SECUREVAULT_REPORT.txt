EXECUTIVE SUMMARY
================================================================================

SecureVault is a production-ready, cross-platform password manager featuring 
post-quantum cryptography, behavioral security, and P2P synchronization. Built 
completely from scratch with custom implementations of NIST-selected quantum-resistant 
algorithms.

This application serves as a drop-in replacement for standard password managers 
(Bitwarden, 1Password, LastPass) while providing superior security through 
quantum-resistant encryption.

1. POST-QUANTUM CRYPTOGRAPHY (RUST CORE)
================================================================================

1.1 ML-KEM (Kyber-768/1024)
------------------------
Location: securevault-core/src/crypto/ml_kem.rs

NIST PQC Round 3 Key Encapsulation Mechanism (KEM)

Parameter        | Kyber-768 | Kyber-1024
----------------|----------|----------
Security Level  | 128-bit  | 256-bit
Public Key      | 1,184 B  | 1,568 B
Secret Key      | 2,400 B  | 3,168 B
Ciphertext      | 1,088 B  | 1,568 B
Shared Secret  | 32 B     | 32 B

Implementation:
- Polynomial ring arithmetic over Z_q[x]/(x^256+1)
- NTT-based polynomial multiplication
- Barrett/Montgomery reduction
- SampleCBD (Centered Binomial Distribution)
- Matrix generation from seed

Key Functions:
  pub fn keygen() -> (PublicKey, SecretKey)
  pub fn encaps(pk: &PublicKey) -> (Ciphertext, SharedKey)
  pub fn decaps(sk: &SecretKey, ct: &Ciphertext) -> SharedKey

1.2 ML-DSA (Dilithium-44/65/87)
-------------------------------
Location: securevault-core/src/crypto/ml_dsa.rs

NIST PQC Round 3 Digital Signature Algorithm

Parameter        | Dilithium-44 | Dilithium-65 | Dilithium-87
----------------|-------------|-------------|-------------
Security Level  | 128-bit     | 192-bit     | 256-bit
Public Key    | 2,592 B    | 4,032 B    | 5,792 B
Secret Key   | 4,000 B    | 6,080 B    | 7,968 B
Signature   | 4,595 B    | 4,895 B    | 4,995 B

Implementation:
- Module-LWE based signatures
- Fiat-Shamir with Aborts
- H1/H3 hash functions (using SHAKE)
- Sample Poly (uniform, Gaussian)
- Power2Round, Decompose

1.3 SPHINCS+ (SHA-256/512)
---------------------------
Location: securevault-core/src/crypto/sphincs.rs

Hash-based signature scheme (stateful)

Parameter        | SPHINCS+-128s | SPHINCS+-256s
----------------|-------------|-------------
Security Level  | 128-bit     | 256-bit
Public Key    | 32 B       | 32 B
Secret Key   | 64 B       | 64 B
Signature   | 7,856 B    | 29,792 B

Implementation:
- FORS (Falcon)
- Merkle Tree (HT)
- WOTS+ (Winternitz)
- SHA-256/512 compression

SHA3-256, SHA3-384, SHA3-512 implementation
